# Architect - アプリ設計支援AI

**曖昧なアイデアを、わずか5分の対話でManus1.5で実装可能な完璧な仕様書に変換。**

Architectは、AI駆動のアプリ設計支援ツールです。開発初心者でも、対話を通じて必要な要件を明確化し、そのままManus1.5エージェントに渡せる詳細な要件定義書を自動生成します。

## 🎯 主な機能

### 📱 対話型ヒアリング
- AIが適切な質問で要件を引き出します
- iOS/Android/Web/PWAなど、プラットフォームの選択をサポート
- 音声入力にも対応

### 🎨 6つのアプリテンプレート
すぐに始められる定番アプリテンプレート:
- 📱 SNS・コミュニティ
- 🛒 ECサイト・オンラインショップ
- ✅ タスク管理・TODO
- 📊 業務管理・ダッシュボード
- 🎓 学習・教育アプリ
- 💰 家計簿・資産管理

### 📄 詳細な要件定義書の自動生成
以下を含む包括的な要件定義書を.mdファイルで出力:
- 機能一覧
- 画面遷移図
- ワイヤーフレーム
- データベース設計
- エッジケース対応
- 非機能要件
- ユーザーストーリー
- UI/UX設計
- 概算クレジット消費

### 🔄 会話履歴の保存と再ダウンロード
- 過去の会話を保存
- いつでも要件定義書を再ダウンロード可能

### 🚀 Manus1.5エージェントモードへのシームレスな連携
- ワンクリックでManus1.5に移動
- 生成された要件定義書をそのまま添付して実装開始

## 🛠️ 技術スタック

### フロントエンド
- **React 19** - UI構築
- **Tailwind CSS 4** - スタイリング
- **shadcn/ui** - UIコンポーネント
- **tRPC** - 型安全なAPI通信
- **Wouter** - ルーティング

### バックエンド
- **Express 4** - サーバーフレームワーク
- **tRPC 11** - エンドツーエンドの型安全性
- **Drizzle ORM** - データベースORM
- **MySQL/TiDB** - データベース

### AI統合
- **Manus LLM API** - 対話型ヒアリングと要件定義書生成
- **Web Speech API** - 音声入力

## 📦 インストールと起動

```bash
# 依存関係のインストール
pnpm install

# データベースのマイグレーション
pnpm db:push

# 開発サーバーの起動
pnpm dev
```

## 🌐 デプロイ

このプロジェクトはManusプラットフォームで開発されており、ワンクリックでデプロイできます。

1. Manusダッシュボードで「Publish」ボタンをクリック
2. 自動的にビルドとデプロイが実行されます
3. カスタムドメインの設定も可能

## 📖 使い方

1. **ログイン**: Manusアカウントでログイン
2. **テンプレート選択または自由入力**: 6つのテンプレートから選ぶか、自由にアイデアを入力
3. **対話**: AIの質問に答えて要件を明確化
4. **ダウンロード**: 「要件定義書をダウンロード」ボタンで.mdファイルを取得
5. **実装**: Manus1.5エージェントモードに要件定義書を添付して「このアプリを作って」と入力
